MQTT客户端Java实现

一、简介

  • 自定义MQTT类继承AbstractMqttClient抽象类
  • 实现自定义处理消息回调方法processMessage
  • 方法解析（可调用函数）
    • init：根据参数初始化MQTT客户端
    • publish：发布主题
    • subscribe：订阅主题（每次订阅成功，会缓存主题信息到topicMap）
    • unsubscribe：取消订阅主题
    • unsubscribeAll：取消所有订阅主题
    • disconnect：断开连接
    • close：关闭客户端

二、源码

提供阿里云物联网平台MQTT客户端工具类、简易的版MQTT服务器（发布、订阅分离）、自定义MQTT服务器客户端
